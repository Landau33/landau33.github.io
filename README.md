## GitHub 个人主页（GitHub Pages）

这个文件夹里的内容可以直接发布成 `https://<你的用户名>.github.io`。

### 如何上线（最短流程）

1. 在 GitHub 新建仓库，仓库名必须是：`<你的用户名>.github.io`
2. 把本文件夹里的所有文件上传到该仓库根目录（不要多套一层文件夹）
3. 等 1-2 分钟后访问：`https://<你的用户名>.github.io`

如果你想用自定义域名，把域名写进仓库根目录的 `CNAME` 文件即可（可选）。

### 个性化内容

- `index.html`：你的主页内容（名字、简介、链接、项目等）
- `assets/css/style.css`：配色和排版
- `assets/js/main.js`：暗色模式、年份等小交互
